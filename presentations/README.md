# 발표 자료

이 폴더는 KMCT 2025년 10월 모임에서 발표된 자료를 보관합니다.

## 발표 목록

발표 자료를 업로드하실 때는 아래 형식으로 목록을 업데이트해 주세요:

### 발표 1: [발표 제목]
- **발표자**: [이름]
- **자료**: [파일명 또는 링크]
- **설명**: [간단한 설명]

### 발표 2: [발표 제목]
- **발표자**: [이름]
- **자료**: [파일명 또는 링크]
- **설명**: [간단한 설명]

## 파일 형식

- PowerPoint (.pptx, .ppt)
- PDF (.pdf)
- Google Slides (링크)
- 기타 문서 형식

## 업로드 가이드

1. 파일명은 `YYYYMMDD_발표자명_주제.확장자` 형식을 권장합니다
   - 예: `20251025_홍길동_Azure소개.pdf`
2. 대용량 파일은 외부 링크를 이용해 주세요
3. 발표 후 가능한 빨리 자료를 공유해 주세요
